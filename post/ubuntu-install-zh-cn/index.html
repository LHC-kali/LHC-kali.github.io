ubuntu安装zh-cn语言

2021年1月23日
| 阅读

ubuntu设置系统语言

apt install locales
apt install fonts-wqy-zenhei #安装中文字体
export LANG=zh_CN.UTF-8 
dpkg-reconfigure locales  #设置语言编码

选择zh_CN.UTF-8.UTF-8 再选择zh_CN.UTF-8即可

为避免语言在重进系统恢复为默认英文设置，可以执行

echo 'export LANG=zh_CN.UTF-8' >> ~/.bashrc

最后再重启linux系统，termux上的ubuntu清除termux后台，再打开termux就行了 这样下次就可以无需每次执行export命令了 上述方法适用于所有基于debian的发行版
