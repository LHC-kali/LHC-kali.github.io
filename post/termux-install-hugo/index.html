termux安装hugo静态博客系统

2021年1月5日
| 阅读

Hugo 是 Go 编写的静态网站生成器，速度快，易用，可配置，Hugo 有一个内容和模板目录，把他们渲染到完全的 HTML 网站，想了解更多hugo信息访问官网

如果本人写的地方有什么不对还请大佬教导一下

安装

①termux安装hugo

pkg i hugo -y

②新建网站

选择你想用的文件夹来做你的hugo网站目录，我这里选择hugo-blog

hugo new site hugo-blog

③给hugo博客安装主题

hugo是没有默认主题的，我们需要从github上找到hugo主题，clone到你的hugo-blog/themes目录下，我这里选择用even主题

cd hugo-blog
git clone https://github.com/olOwOlo/hugo-theme-even themes/even

④激活even主题

把even/exampleSite下的config.toml文件复制的hugu-blog目录下

cp themes/even/exampleSite/config.toml hugo-blog

⑤注意：对于这个主题，你应该使用 post 而不是 posts，新建文章命令

hugo new post/some-content.md

⑥启动hugo服务器

hugo server

默认访问地址是http://localhost:1313,自定义端口方法

hugo server -D -p 8080
